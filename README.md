# Myntra HackerRamp – Fashion E-Commerce Platform

A full-stack fashion e-commerce web application built for **Myntra HackerRamp**, featuring product browsing, user authentication, cart management, and order placement.

## 🚀 Features
- 🛍️ Browse and search fashion products
- 🔐 User registration & login with JWT authentication
- 🛒 Add to cart and manage orders
- 💳 PayPal payment integration
- 👤 User profile management
- 🔧 Admin panel for product & order management
- 📱 Fully responsive UI

## 🛠️ Tech Stack
- **Frontend:** React JS, Redux, React Bootstrap
- **Backend:** Node JS, Express JS, MongoDB Atlas
- **Auth:** JWT Authentication
- **Tools:** Git, GitHub, Postman

## ⚙️ Getting Started

### Installation
```bash
git clone https://github.com/Malikabrar01/Myntra-Hackerramp-project.git
cd Myntra-Hackerramp-project
npm install
cd frontend && npm install
```

### Run
```bash
# Terminal 1 - Backend
npm start

# Terminal 2 - Frontend
cd frontend
NODE_OPTIONS=--openssl-legacy-provider npm start
```
